  <how-to-verify>
Start the app if not already running:
```bash
npm run dev
```
Open http://localhost:5173 in your browser.

**Verification checklist (check each one):**

1. **VIEW-01 — Markdown renders correctly**
   - Click a .md file that contains at least one of: a table, a fenced code block, GFM checkboxes (`- [ ] item`), or headings
   - Verify: the preview shows rendered HTML output — not raw markdown text
   - Verify: fenced code block shows syntax-highlighted output (not plain monospace)
   - Verify: YAML frontmatter (if present) does NOT appear in the preview

2. **VIEW-02 — Multiple tabs open simultaneously**
   - Click three different files in the sidebar one by one
   - Verify: three tabs appear in the tab bar, each showing the filename
   - Click a file that is already open — verify: focus switches to its existing tab (no duplicate tab created)

3. **VIEW-03 — Tab switching**
   - With multiple tabs open, click each tab in turn
   - Verify: the displayed content matches the file name shown in the active tab

4. **VIEW-04 — Tab closing**
   - Hover over a tab — verify the close (x) button appears
   - Click close on the active tab — verify: it disappears and an adjacent tab becomes active
   - Close all tabs — verify: WelcomeScreen appears

5. **DSNG-01 — Frosted glass content panel**
   - With a file open, look at the main content card
   - Verify: the panel has a soft frosted/translucent appearance — you can sense the cream background through it (subtle white/translucent fill, not solid white)

6. **DSNG-02 — Orange accent palette**
   - Active tab: verify orange underline at the bottom of the active tab
   - Selected file in tree: verify orange highlight on the currently-open file
   - Headings in preview: verify h1/h2/h3 text appears in orange (via prose-orange)
   - TOC active item: scroll the document and verify the active heading in the TOC turns orange

7. **DSNG-03 — Big Hero 6 aesthetic**
   - Stand back and look at the overall layout
   - Verify: background is warm cream (not cold white, not dark)
   - Verify: the app feels bright, clean, and warm — not clinical or generic
   - Verify: orange accents are present but not overwhelming

8. **DSNG-04 — Airy typography**
   - Open a long markdown document
   - Verify: text is comfortable to read — generous line-height, comfortable margins, no cramped walls of text
   - Verify: no visual noise — unnecessary elements don't compete for attention

**Tab drag-to-reorder (bonus check):**
   - With 2+ tabs open, drag one tab to a different position
   - Verify: tab order changes and the dragged tab settles in the new position

**TOC panel:**
   - Open a file with multiple headings (h1, h2, h3)
   - Verify: the right panel shows a "On this page" list
   - Scroll down in the content — verify: the active heading in the TOC updates as you scroll
  </how-to-verify>
