  <action>

   ```bash
   npm install chokidar --workspace=server
   ```

2. Create `server/src/lib/watcher.ts`:
   - Class `FileWatcherService` with a private `FSWatcher` and a private `Set<Subscriber>`
   - Constructor takes `rootDir: string`; initialises chokidar with `ignoreInitial: true` and `awaitWriteFinish: { stabilityThreshold: 50, pollInterval: 10 }`
   - On `change` event: compute `relativePath = absPath.replace(rootDir + '/', '')`, call all subscribers with `{ type: 'change', path: relativePath }`
   - On `add` event: same path computation, call all subscribers with `{ type: 'add', path: relativePath }`
   - `subscribe(cb)`: adds `cb` to the Set, returns an unsubscribe function `() => this.subscribers.delete(cb)`
   - `async close()`: awaits `this.watcher.close()`
   - Export the class as a named export
   - Type: `type Subscriber = (event: { type: 'change' | 'add'; path: string }) => void`

   - Sets headers: `Content-Type: text/event-stream`, `Cache-Control: no-cache`, `Connection: keep-alive`
   - Calls `res.flushHeaders()` immediately
   - Subscribes to `fastify.fileWatcher`
   - On subscriber callback: writes `event: {event.type}\ndata: ${JSON.stringify({ path: event.path })}\n\n` to `res`
   - On `req.raw.on('close')`: calls unsubscribe
   - Returns `await new Promise<void>(resolve => req.raw.on('close', resolve))`
